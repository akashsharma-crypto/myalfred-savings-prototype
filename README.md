# myAlfred — Savings Plan Prototype

Interactive, clickable prototype of the end-to-end Savings Plan journey for the myAlfred app, matching the myAlfred design system.

## Flow

Home → Calculator → Quotes (multi-insurer) → Fund Selection (Sukoon) → Upload Documents → Verify Details → Sign & Pay → Confirmation

Tap through the phone mockup, or use the pill buttons below it to jump to any screen.

## Deploy

This is a static single-page HTML file — no build step needed. Deploy directly on Vercel:

1. Push this repo to GitHub
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import this repo — Vercel will detect it as a static site automatically
4. Deploy

No environment variables or framework config required.
